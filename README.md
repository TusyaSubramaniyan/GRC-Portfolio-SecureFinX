## 🏢 Client Overview

SecureFinX is a simulated 60-person fintech startup based in 
Chennai, India. They operate a mobile payment application with 
200,000 active users, storing Aadhaar-linked KYC data and 
payment transaction history on AWS Mumbai.

They engaged me as their GRC Consultant to:
- Assess their current security posture
- Identify and score information security risks
- Build their ISO 27001-aligned ISMS from scratch
- Write core security policies
- Conduct an internal audit and produce a formal report
